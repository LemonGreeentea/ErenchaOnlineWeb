비공식 DLC 사용 설명서

아래 안내를 따라 에렌샤 온라인에서 비공식 DLC를 안전하고 쾌적하게 사용해 주세요.

  1. 상단 탭에서 필요한 지역, 캐릭터, 컨텐츠, 이벤트 등을 선택한 뒤 다운로드 페이지에서 로어북 JSON 및(또는) 에셋을 생성하여 내려받아 적용합니다.
  2. 과도한 DLC 선택은 대화 흐름과 모델 응답 품질에 영향을 줄 수 있습니다. 사용 환경에 맞춰 필요한 항목만 선별 적용하시고, 필요 시 선택을 줄여 다시 생성해 주시기 바랍니다.
  3. 각 DLC의 상세한 세계관·등장인물·사용 가이드는 해당 DLC에 기재된 출처를 참고해 주세요. 출처 문서를 확인하시면 더 정확한 정보와 주의사항을 보실 수 있습니다.
  4. 본 프로젝트에 관심을 가져 주셔서 감사합니다. 에렌샤 온라인 봇 제작자와 개별 DLC를 제작해 주신 분들께 감사의 댓글이나 피드백을 남겨 주시면 큰 힘이 됩니다.
