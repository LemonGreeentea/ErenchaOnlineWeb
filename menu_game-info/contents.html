🛠 The System — 스탯 대신 숙련, 그리고 보정

에렌샤에는 스탯 분배가 없습니다. 의도한 행동이 곧 성장입니다. 검을 휘두르면 한손검, 주문을 외우면 계열 마법, 회피에 성공하면 스텝 숙련이 오릅니다. 숙련이 높아질수록 시스템은 동작 궤적을 미세하게 보정해, 같은 힘으로 더 예리하고 안정적인 결과를 냅니다.

  • 이중 성장: 캐릭터 레벨 EXP와 숙련 EXP 동시 획득
  • 그레이드 업: 숙련 Lv.100 → 레벨 0으로 초기화 + 영구 보너스
  • 연계성: 여러 숙련의 조합은 히든 피스/직업의 단서가 될 수 있음

✨ 스킬 시스템

중요한 건 ‘어떤 스킬을 가졌나’보다 ‘어떻게 엮어 쓰나’입니다. 환경과의 상호작용으로 전술을 확장해 보세요—강을 얼려 길을 만들고, 숲을 태워 퇴로를 차단하는 식으로요. 때로는 시스템의 빈틈을 파고든 플레이가 ‘시그니처 스킬’로 정착해 새로운 법칙이 되기도 합니다.

🚀 다채로운 전직 시스템

직업에 갇힐 필요가 없습니다. 언제든 무직으로 돌아갔다가 조건을 채우면 즉시 다른 전직 퀘스트를 받을 수 있습니다. 일단 전직했던 직업은 제약 없이 다시 되찾을 수 있고, 테마에 맞는 숙련 보너스로 더 높은 단계로 뛸 수 있습니다. 여러 숙련을 함께 올리면 히든 전직의 단서를 만날지도 모릅니다.
